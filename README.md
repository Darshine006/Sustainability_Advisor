
🌱 Sustainability Advisor



A web-based tool promoting environmentally responsible decision-making.
Translates sustainability principles into actionable guidance for users, encouraging eco-friendly habits through a simple, accessible interface.

🔹 Features

Provides practical sustainability recommendations for daily decisions

Lightweight, fast-loading web interface

Encourages environmentally responsible behavior

Easily accessible via browser deployment

🎯 Objective

Developed as part of the AI for Sustainability initiative by 1M1B, this project demonstrates how digital tools can raise awareness and guide environmentally conscious choices.

Key goals:

Promote sustainability awareness through a web interface

Provide a prototype for AI-driven advisory systems

Encourage eco-friendly decision-making

🛠️ Tech Stack

Frontend: HTML, CSS

Deployment: GitHub Pages

Version Control: Git & GitHub
